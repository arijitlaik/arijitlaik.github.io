Running Underworld2 natively on Snellius/LISA

Link to script to compile underworld natively -> here Copy this script to the home directory and then run it as source

source install_underworld2_snellius_lisa_2021.sh

Content of the script

module load 2021 PETSc/3.15.1-foss-2021a  HDF5/1.10.7-gompi-2021a h5py/3.2.1-foss-2021a SWIG/4.0.2-GCCcore-10.3.0
pip3 install --user lavavu pint
cd cd /home/$USER
git clone https://github.com/underworldcode/underworld2.git
cd underworld2/underworld/libUnderworld
./configure.py --with-debugging=0
./compile.py -j8

Following are the changes to the jobscript:

  1. Do not copy the singularity file to the temp directory

    Singularity:

    echo "********** Copying $IMAGE_STORE/underworld2_$IMAGE_VERSION.sif to  ${SCRATCH_SHARED_DIR} **********"
    cp $IMAGE_STORE/underworld2_$IMAGE_VERSION.sif ${SCRATCH_SHARED_DIR}
    echo "********** Copying Singularity Image Done **********"

    Native:

    # echo "********** Copying $IMAGE_STORE/underworld2_$IMAGE_VERSION.sif to  ${SCRATCH_SHARED_DIR} **********"
    # cp $IMAGE_STORE/underworld2_$IMAGE_VERSION.sif ${SCRATCH_SHARED_DIR}
    # echo "********** Copying Singularity Image Done **********"

  2. Add modules to load and append PYTHONPATH

    Singularity:

    srun -n 256 singularity exec --pwd $PWD underworld2_$IMAGE_VERSION.sif  python3 r3dB_mlow_sn.py

    Native:

    module load 2021 PETSc/3.15.1-foss-2021a  HDF5/1.10.7-gompi-2021a h5py/3.2.1-foss-2021a SWIG/4.0.2-GCCcore-10.3.0
    export PYTHONPATH=/home/$USER/underworld2:/home/$USER/underworld2/underworld/libUnderworld/build/lib:$PYTHONPATH
    srun -n 256 python3 script.py
